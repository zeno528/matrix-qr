# 订阅链接二维码生成器 - 规范文档

## 1. 项目概述

- **项目名称**: 订阅链接二维码生成器
- **项目类型**: 单页面 Web 应用
- **核心功能**: 将代理订阅链接转换为二维码，方便移动设备扫码导入
- **目标用户**: 使用代理软件（Clash、Surge、Shadowrocket、Quantumult等）的用户

## 2. UI/UX 规范

### 布局结构

- 单页面垂直布局
- 居中显示，最大宽度 500px
- 页面结构：标题 → 输入区域 → 生成按钮 → 二维码显示区

### 视觉设计

**配色方案**
- 背景色: `#0a0a0f` (深邃黑)
- 卡片背景: `#12121a` (暗灰)
- 主色调: `#00d4aa` (青绿色，象征科技感)
- 辅助色: `#6366f1` (靛蓝色)
- 文字主色: `#e4e4e7`
- 文字次色: `#71717a`
- 边框色: `#27272a`

**字体**
- 主字体: `"JetBrains Mono", "Fira Code", monospace`
- 标题: 24px, font-weight: 700
- 正文: 14px

**间距**
- 页面内边距: 24px
- 元素间距: 16px
- 输入框内边距: 14px

**视觉效果**
- 卡片圆角: 12px
- 输入框聚焦时发光效果 (box-shadow: 0 0 0 2px rgba(0, 212, 170, 0.3))
- 按钮悬停渐变背景
- 二维码容器 subtle glow 效果

### 组件规范

**输入框**
- 宽度: 100%
- 高度: 120px (多行文本域)
- 背景: `#1a1a24`
- 边框: 1px solid #27272a
- 占位符提示: "粘贴 Clash / Surge / Shadowrocket 订阅链接..."

**生成按钮**
- 宽度: 100%
- 高度: 48px
- 背景: 渐变 linear-gradient(135deg, #00d4aa, #6366f1)
- 圆角: 8px
- 悬停: scale(1.02), 亮度提升

**二维码显示区**
- 白色背景 padding: 16px
- 居中显示二维码
- 最大尺寸: 280px × 280px
- 生成时 fade-in 动画

## 3. 功能规范

### 核心功能

1. **链接输入**
   - 支持多行输入（可一次性输入多个订阅链接）
   - 自动去除首尾空白
   - 检测非空输入

2. **二维码生成**
   - 使用 qrcode.js 库生成二维码
   - 支持任意文本/链接
   - 二维码尺寸: 256px
   - 容错级别: M (15%)

3. **多链接处理**
   - 逐行解析输入
   - 为每个有效链接生成独立二维码
   - 显示行号和链接标识

4. **复制功能**
   - 点击二维码下方显示原文链接
   - 可复制原始链接

### 用户交互

1. 用户粘贴/输入订阅链接
2. 点击"生成二维码"按钮
3. 页面显示对应的二维码
4. 移动设备扫码即可导入

### 边缘情况处理

- 空输入: 显示提示"请输入订阅链接"
- 无效链接: 仍生成二维码（不对内容做校验）
- 过长链接: 自动适应二维码大小

## 4. 技术实现

- 纯 HTML + CSS + JavaScript
- 使用 CDN 引入 qrcode.js
- 无需后端服务
